CONTEXTO INSTITUCIONAL – MANUAL DE IDENTIDAD VISUAL SENA
VERSIÓN 2024
USO OBLIGATORIO PARA TODAS LAS RESPUESTAS DE LA IA

Este proyecto pertenece al Servicio Nacional de Aprendizaje – SENA (Colombia).
CUALQUIER salida generada por el asistente de IA (código, vistas, estilos, UI, textos, componentes, mockups o documentación visual)
DEBE cumplir estrictamente con el Manual de Identidad Visual SENA 2024.

================================================================
1. PRINCIPIOS GENERALES
================================================================
- El diseño debe ser institucional, sobrio, accesible y claro.
- No usar estilos llamativos, comerciales o ajenos al sector público.
- Priorizar legibilidad, contraste y accesibilidad.
- Evitar saturación de colores, efectos visuales innecesarios o modas UI no institucionales.
- El software representa una entidad del Estado colombiano.

================================================================
2. COLORES INSTITUCIONALES (OBLIGATORIOS)
================================================================

COLOR PRINCIPAL SENA (USO PRIORITARIO):
- Verde institucional:
  HEX: #39A900
  RGB: 57, 169, 0

Este color se usa para:
- Elementos principales
- Botones primarios
- Indicadores de acción
- Íconos institucionales
- Estados activos

COLORES SECUNDARIOS AUTORIZADOS:
- Verde oscuro: #007832
- Azul oscuro: #00304D
- Azul claro: #50E5F9
- Violeta: #71277A
- Amarillo: #FDC300

COLORES NEUTROS:
- Blanco: #FFFFFF
- Gris claro: #F6F6F6
- Negro: #000000

REGLAS:
- No inventar colores nuevos.
- No usar degradados llamativos.
- El verde SENA NO debe usarse para marcas externas.
- El verde siempre debe contrastar adecuadamente.

================================================================
3. USO DEL COLOR EN INTERFACES (UI)
================================================================
- Fondo predominante blanco o gris claro.
- Verde SENA para acciones primarias.
- Colores secundarios solo como apoyo visual.
- Nunca saturar la interfaz con verde.
- Preferir jerarquía visual clara antes que decoración.

Ejemplo recomendado:
- 80% blanco / gris claro
- 20% verde institucional

================================================================
4. TIPOGRAFÍA INSTITUCIONAL
================================================================

TIPOGRAFÍA PRINCIPAL:
- Work Sans (todas las vistas, interfaces y UI)

Pesos recomendados:
- Regular
- Medium
- SemiBold
- Bold

NO usar:
- Thin
- ExtraLight

TIPOGRAFÍA SECUNDARIA (opcional en web):
- Calibri (solo si es necesario)

REGLAS:
- Texto siempre legible.
- Alto contraste con el fondo.
- No usar tipografías decorativas o comerciales.

================================================================
5. LOGO SENA (REGLAS CRÍTICAS)
================================================================
- El logo NO debe modificarse.
- No cambiar proporciones.
- No aplicar sombras, efectos, biseles ni degradados.
- Solo usar en:
  - Verde institucional
  - Blanco
  - Negro

- Siempre respetar el área de seguridad.
- No rotar.
- No encerrar en contenedores arbitrarios.

================================================================
6. ICONOGRAFÍA
================================================================
- Íconos de línea (outline).
- Sin relleno.
- Grosor consistente.
- Estilo minimalista.
- Evitar íconos comerciales o excesivamente detallados.

================================================================
7. FOTOGRAFÍA E IMÁGENES (SI APLICA)
================================================================
- Imágenes reales, no artificiales.
- Mensaje positivo, institucional y humano.
- Mostrar formación, trabajo, innovación y talento.
- No usar imágenes negativas o sensacionalistas.
- No colocar texto sobre imágenes con bajo contraste.

================================================================
8. ESTILO VISUAL PARA SOFTWARE Y VISTAS
================================================================
Cuando se solicite:
- crear una vista
- diseñar una interfaz
- generar HTML, Blade, CSS o componentes UI

La IA DEBE:
- Respetar el color verde SENA como eje
- Mantener diseño limpio
- Priorizar claridad funcional
- Usar estructuras simples y ordenadas
- Evitar animaciones innecesarias

================================================================
9. CONTEXTO LEGAL E INSTITUCIONAL
================================================================
- El diseño debe cumplir la Ley 2345 de 2023.
- No usar marcas de gobierno diferentes al SENA.
- No crear logotipos alternos.
- No alterar identidad institucional.

================================================================
10. INSTRUCCIÓN FINAL PARA LA IA
================================================================
ANTES de responder cualquier solicitud:
- Verifica que la solución cumpla el Manual de Identidad SENA.
- Si una petición contradice el manual, ajusta la respuesta automáticamente.
- Prioriza siempre coherencia institucional sobre preferencias estéticas.
